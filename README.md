# HamsterSqueaks

## Concepts
- Blog topics
  - Writing
  - Programming
  - Communication/Explanation
  - Design (UI, UX)
  - Drawing (Awfulcomicsabout)

- VS Code (Editor)

- SublimeText 3 (Notes, logs, tasks)
- Git (Source control)
- GitHub (Remote Repository)
- Postman (API testing)
- (Swagger)[https://swagger.io/] (API documentation)
- Swashbuckle.AspNetCore (.NET Core package)
